【開き方】
  index.html をダブルクリック、または
  任意のブラウザ（Chrome / Edge / Firefox）でファイルを開いてください。

  推奨ブラウザ: Google Chrome / Microsoft Edge

【操作方法】
  → / スペースキー   : 次のスライドへ
  ←               : 前のスライドへ
  上部のドット       : 任意のスライドに直接ジャンプ
  タッチスワイプ      : スマートフォン / タブレットでも操作可能

【スライド構成】
  1. タイトル
  2. 課題  - リモートワークの「あるある問題」
  3. ソリューション - 猫で可視化
  4. 仕組み - アーキテクチャフロー
  5. 猫の5状態 - idle / typing / clicking / scrolling / sleeping
  6. 機能一覧
  7. チーム全体像デモ
  8. 技術スタック
  9. CTA / エンディング

【注意事項】
  - インターネット接続不要（Google Fonts のみ使用）
  - オフライン環境では日本語フォントがシステムフォントにフォールバックします
